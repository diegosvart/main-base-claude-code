## Architecture pattern

**Screaming Architecture + Explicit Ports**

```
app/
├── [domain-1]/          ← domain visible in folder name
│   ├── router.py        ← HTTP layer only (FastAPI)
│   ├── service.py       ← pure business logic — only imports ports.py
│   ├── schemas.py       ← Pydantic request/response contracts
│   └── ports.py         ← abstract interfaces (ABC) — only if >1 external dep
│
├── [domain-2]/
│   └── ...
│
├── core/
│   ├── config.py        ← settings (pydantic-settings)
│   ├── security.py      ← auth utilities
│   ├── db.py            ← async engine + session factory
│   └── dependencies.py  ← FastAPI Depends
│
├── infra/               ← concrete implementations of all ports
│   ├── db/              ← SQLAlchemy repositories
│   ├── cache/           ← Redis implementations
│   └── email/           ← SMTP implementations
│
└── main.py              ← FastAPI app, lifespan, middleware
```
